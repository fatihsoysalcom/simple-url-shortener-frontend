# Simple URL Shortener Frontend

This example demonstrates the frontend JavaScript logic for a URL shortener. It shows how to handle form submissions, make asynchronous requests to a hypothetical backend API, and display results or errors to the user.

## Language

`html`

## How to Run

Save this code as `index.html` and open it in a web browser. Note that the `fetch` request to `/shorten` will fail unless a backend server is running at that endpoint.
